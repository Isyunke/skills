## 🎯 这是什么？

**简历炼金术士**是一个帮助你：

1. **深挖自己的能力**——通过对话式引导，挖掘你都没意识到的闪光点
2. **针对性优化简历**——根据每个 JD 定制简历，不是一份简历投天下
3. **系统性准备面试**——从简历到面试，形成完整的求职闭环
4. **持续性能力提升**——发现技能缺口，制定学习计划

## ⚖️ 三条不可妥协原则

### 1. 真实性原则 (Truth First)

> 简历内容必须基于真实经历，绝不能虚构或夸大。

- 所有经历必须有 project.md 的证据链
- 简历生成时自动校验：每项技能必须有对应项目支撑
- 用户试图添加未验证的内容时，强制提示"请先补充相关经历"

### 2. 针对性原则 (Tailored for You)

> 每份简历必须针对特定 JD 优化，不能一份简历投天下。

- 每份简历必须关联一个 JD（岗位文档）
- 简历生成时自动匹配：JD 关键词 vs 简历内容
- 用户试图"一份简历投所有"时，强制提示"请为这个 JD 定制简历"

### 3. 证据链原则 (Evidence Chain)

> 简历中的每项技能必须有项目/经历支撑，不能空谈。

- 简历生成时自动检查：每项技能 → 对应项目 → 具体成果
- 缺少证据链的技能自动降级为"了解"或移除
- self-profile.md 中的技能必须标注证据强度

## 🚀 快速开始

### 第一步：初始化

在你的求职项目目录中，打开 Claude Code 并说：

```
初始化简历炼金术士
```

或

```
/init
```

系统会问你 4 个问题：
1. 你想从事什么职位？
2. 你目标什么行业？
3. 你有多少年经验？
4. 你有什么现成的简历或项目文档吗？

### 第二步：深挖经历

```
聊聊我的经历
```

系统会用 STAR 法则引导你挖掘真实经历：
- **S**ituation：项目背景是什么？
- **T**ask：你的任务是什么？
- **A**ction：你做了什么？
- **R**esult：结果如何？（量化！）

### 第三步：分析 JD

```
分析这个 JD [粘贴 JD 文本]
```

系统会自动解析 JD，生成：
- 岗位分析
- 技能匹配度
- 优化建议
- 学习计划

### 第四步：生成简历

```
针对这个 JD 优化简历
```

系统会根据你的经历和 JD，生成针对性简历。

### 第五步：导出 PDF

```
导出 PDF
```

系统会将 HTML 简历转换为 PDF，用于投递。

## 🤔 FAQ

### Q: 这个工具和 ChatGPT 有什么区别？

A: ChatGPT 是通用助手，会告诉你"怎么写简历"。
简历炼金术士是你的专属求职顾问，会帮你：
1. 深挖你的真实经历
2. 针对每个 JD 定制简历
3. 准备面试
4. 发现技能缺口

### Q: 我没有项目经历怎么办？

A: 通过 `/resume-mine` 引导你挖掘：
- 课程项目
- 个人项目
- 实习经历
- 工作经历

### Q: 简历内容会造假吗？

A: 不会。简历炼金术士有三条不可妥协原则：
1. 真实性原则：只记录真实经历
2. 针对性原则：针对 JD 优化
3. 证据链原则：每项技能必须有项目支撑

### Q: 如何保证简历质量？

A: 简历炼金术士有盲评估机制：
- 独立评估简历质量
- 检查关键词覆盖率
- 检查量化数据充分度
- 检查 STAR 法则完整度
