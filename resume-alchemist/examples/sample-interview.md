## 技术问题（根据 JD）

### 分布式系统

**Q: 什么是分布式系统？**

**回答要点**：
- 定义：多个独立计算机协作完成任务
- 优势：可扩展、高可用
- 挑战：一致性、网络分区

**Q: CAP 理论是什么？**

**回答要点**：
- C (Consistency): 一致性
- A (Availability): 可用性
- P (Partition tolerance): 分区容错
- 三者只能满足其二

**Q: Raft 协议是什么？**

**回答要点**：
- 领导者选举
- 日志复制
- 安全性

---

### 数据库优化

**Q: 如何优化数据库查询？**

**回答要点**：
1. 添加索引
2. 优化 SQL
3. 使用缓存
4. 分库分表

**Q: 什么是慢查询？如何排查？**

**回答要点**：
- 定义：执行时间超过阈值的查询
- 排查：开启慢查询日志，分析执行计划

---

### 缓存策略

**Q: Redis 和 Memcached 的区别？**

**回答要点**：
- Redis: 支持多种数据结构，持久化
- Memcached: 简单 KV，多线程

**Q: 缓存穿透、击穿、雪崩是什么？**

**回答要点**：
- 穿透：查询不存在的数据
- 击穿：热点 key 过期
- 雪崩：大量 key 同时过期
