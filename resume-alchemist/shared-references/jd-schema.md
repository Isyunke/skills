## 字段说明

### 元数据

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| JD ID | string | ✅ | jd-<NNN>，唯一标识 |
| 公司 | string | ✅ | 公司名称 |
| 职位 | string | ✅ | 职位名称 |
| 薪资 | string | ❌ | 薪资范围 |
| 地点 | string | ❌ | 工作地点 |
| 分析时间 | ISO 8601 | ✅ | 分析时间 |

### 核心要求

| 字段 | 类型 | 说明 |
|---|---|---|
| 要求 | string | JD 中的要求 |
| 优先级 | enum | "必须" / "最好" / "加分" |
| 匹配度 | enum | "✅ 强匹配" / "⚠️ 弱匹配" / "❌ 无匹配" |
| 证据 | string | 你的项目/经历 ID |
| 缺口 | string | 需要补充的内容 |

### 技能匹配度

| 字段 | 类型 | 说明 |
|---|---|---|
| 强匹配 | percentage | 有强证据支撑的技能 |
| 弱匹配 | percentage | 有弱证据支撑的技能 |
| 无匹配 | percentage | 无证据支撑的技能 |

## 更新规则

- **不覆盖**：每次更新 append 到文件末尾
- **原子写**：使用 .tmp → rename
- **git 版本控制**：用户可回滚

## 与简历的关系

jd-analysis.md 是简历优化的**指导**：

| jd-analysis.md | 简历 |
|---|---|
| 核心要求 | 简历重点 |
| 优化建议 | 简历调整方向 |
| 面试重点 | 面试准备方向 |
| 关键词列表 | 简历关键词 |

**原则**：简历必须针对 JD 优化，不能一份简历投天下。
