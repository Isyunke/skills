## 字段说明

### 基本信息

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 姓名 | string | ✅ | 真实姓名 |
| 目标职位 | string | ✅ | 从 state 派生 |
| 目标行业 | string | ✅ | 从 state 派生 |
| 经验水平 | string | ✅ | 从 state 派生 |
| 教育背景 | string | ❌ | 学校+专业+学历 |

### 技能树

| 字段 | 类型 | 说明 |
|---|---|---|
| 技能 | string | 技能名称 |
| 熟练度 | enum | "了解" / "熟练" / "精通" |
| 评分 | int/10 | 0-10 分 |
| 证据 | string | 项目 ID 或工作经历 ID |
| 最后使用 | YYYY.MM | 最后一次使用时间 |

### 证据强度

| 强度 | 含义 | 简历处理 |
|---|---|---|
| 🔴 无证据 | 技能声明，但无项目支撑 | 不出现在简历 |
| 🟡 弱证据 | 有项目，但贡献不明确 | 标"了解" |
| 🟢 强证据 | 有项目，有量化成果 | 可写"熟练/精通" |

## 更新时机

- `/resume-mine` 完成后自动更新
- 用户主动说"更新技能树"时
- `/resume-learn` 完成学习后更新

---

## 更新规则

- **增量更新**：不覆盖，只追加或修改
- **证据校验**：新增技能必须有项目支撑
- **原子写**：使用 .tmp → rename
- **git 版本控制**：用户可回滚

---

## 与简历的关系

self-profile.md 是简历的**数据源**：

| self-profile.md | 简历 |
|---|---|
| 基本信息 | 简历头部 |
| 个人总结 | 自我评价 |
| 技能树 | 技能列表 |
| 工作经历 | 工作经历 |
| 项目经历 | 项目经历 |
| 教育背景 | 教育背景 |

**原则**：简历内容必须来自 self-profile.md，不能凭空编写。
